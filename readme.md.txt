# Sweet Potato (OFSP) Regeneration Calculator

This is a web-based simulation tool to model the generational growth, cost, and nutritional output of Orange-Fleshed Sweet Potato (OFSP) cultivation. It is based on the spreadsheet model developed by Tim Maurer and [Your Name].

The application is built with vanilla HTML, CSS, and JavaScript, requiring no server or database to run.

## How to Run Locally

1.  **Download:** Make sure you have all three files (`index.html`, `style.css`, `app.js`) in the same folder on your computer.
2.  **Run:** **Double-click the `index.html` file.**
3.  **Use:** It will open in your default web browser. You can now change any input and click "**Run Simulation**" to see the results instantly.

## How to Deploy to GitHub & GitHub Pages (Make it Live)

You can host this application live on the web for free using **GitHub Pages**.

1.  **Create a GitHub Repository:**
    * Log in to [GitHub](https://github.com).
    * Create a **new repository** (e.g., `ofsp-calculator`).
    * Make sure it is **Public**.

2.  **Upload Your Files:**
    * In your new repository, click the "**Add file**" button and then "**Upload files**."
    * Drag and drop `index.html`, `style.css`, and `app.js` (and `README.md`) into the uploader.
    * Click "**Commit changes**."

3.  **Activate GitHub Pages:**
    * In your repository, go to the "**Settings**" tab.
    * In the side menu, click on "**Pages**."
    * Under "Build and deployment," set the **Source** to "Deploy from a branch."
    * Under "Branch," select `main` (or `master`) and leave the folder as `/ (root)`.
    * Click "**Save**."

4.  **Go Live!**
    * Wait about 1-2 minutes.
    * Refresh the "Pages" settings screen. You will see a green box at the top with your live URL: `https://YOUR-USERNAME.github.io/ofsp-calculator`
